class compass.db.model.ClusterState(**kwargs)¶

Bases: sqlalchemy.ext.declarative.api.Base

The state of the Cluster.

id: int, identity as primary key. state: Enum, ‘UNINITIALIZED’: the cluster is ready to setup.

‘INSTALLING’: the cluster is not installing. ‘READY’: the cluster is setup. ‘ERROR’: the cluster has error.
